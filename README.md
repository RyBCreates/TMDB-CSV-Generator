# TMDB Flashcard Dataset Exporter

Exports movie data from TMDB into a CSV for flashcard generation.

## Data Included

- Movie title
- Release year
- Director
- Cinematographer
- Top 5 actors
- TMDB rating

## Setup

1. Install dependencies

npm install

2. Create .env file

TMDB_KEY=your_api_key

3. Run script

node exportMovies.js
